● Now I understand aipack. Key points:

  What it is: CLI tool for AI-powered code generation using markdown agent files
   (.aip). Multi-stage pipeline with Lua scripting and Handlebars templating.

  Role in your process (from PROJECT_PROCESS.md):
  - Session 4: Claude Code orchestrates, aipack handles repetitive file-by-file
  code generation
  - Reduces context load on Claude for bulk operations

  Critical for Claude agents: Must use -s flag (single-shot/non-interactive
  mode):
  aip run pro@coder -s

  The workflow from your prompting guide:
  1. Create/edit prompt file with write_mode: false
  2. Run aip run pro@coder -s to preview
  3. Review output
  4. Set write_mode: true, re-run to write files
  5. Git commit + verify
